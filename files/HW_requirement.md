# Homework Submission Requirements

## 1. File Format
- **Submit only ONE `.zip` file.**
- Zip **all files** into a single `.zip` file.
- Name the zip file using the format: `HW#_YourName.zip`  
  - Example: `HW1_BobGoodman.zip`
- The zip file **must include all source files and all screenshots**.

## 2. Program Requirements
- Ensure your program **compiles and runs successfully**.
- Include **screenshots of your programâ€™s output**.
  - **All screenshots must clearly show terminal lines**, demonstrating that the output was captured directly from the terminal.
  - Screenshots of manually typed, edited, or fabricated terminal output are **not accepted**.
- **Code must be properly indented** for readability.
- **Include comments** in your code to explain the logic and important steps.

## 3. Grading Notes
- Failure to follow **any** of these instructions will result in a **score of 0**.
- **Late submissions:**
  - **10 points deducted per day** late.
  - Submissions **10 days late or more** will receive **0 points**.
