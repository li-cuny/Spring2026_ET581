# Java Math Class & Random Numbers – Exercises

## Exercise 1 – Using `Math.max()` and `Math.min()`
Write a Java program that prints the **larger** and **smaller** number between `15` and `27`.

Example output:
```
Max: 27
Min: 15
```

---

## Exercise 2 – Absolute Value
Write a program that finds the **absolute value** of `-42`.

Expected output:
```
Absolute value: 42
```

---

## Exercise 3 – Square Root
Write a program that calculates the **square root of 81** using the `Math` class.

Expected output:
```
Square root: 9.0
```

---

## Exercise 4 – Power
Write a program that calculates:


5^3


using `Math.pow()`.

Expected output:
```
Answer: 125.0
```
