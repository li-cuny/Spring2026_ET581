# Car Class Definition

Create a class called `Car` with the following **member variables**:

- `year` (int)
- `model` (String)
- `color` (String)

Complete the following tasks:

1. Create a **method** called `printInfo()` that prints the car information.

2. Create two **constructors**:
   - A constructor with **one parameter (`year`)**
   - A constructor with **three parameters (`year`, `model`, `color`)**

3. In the `Main` class:
   - Create one `Car` object using the **one-parameter constructor**.
   - Create another `Car` object using the **three-parameter constructor**.
   - Call the `printInfo()` method for both objects.

4. Change the attributes `year`, `model`, and `color` to **private**.  
   Try accessing them directly from `Main` and observe what happens.
